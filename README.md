# Azure Infrastructure Admin Lab

Hands-on Azure infrastructure project focused on core administration skills including virtual machines, networking, access control, monitoring, backup, and infrastructure as code with Terraform.

## Objectives

- Deploy and manage Windows and Linux VMs in Azure
- Configure networking with VNets, subnets, and NSGs
- Implement access control with Microsoft Entra ID and RBAC
- Enable monitoring and backup for core resources
- Rebuild the environment using Terraform
